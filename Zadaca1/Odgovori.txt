Što znači kratica CSS?
Cascading Style Sheets.

Od čega se sastoji jedno CSS pravilo (CSS rules)?
Sastoji se od selektora i deklaracije (svojstvo i vrijednost).

Što su CSS selektori i koji su najbitniji?
CSS selektori su elementi CSS-a kojima označavamo pojedine tagove, klase ili id-jeve kako bi ih mogli urediti.
Najbitniji su TAG, CLASS i ID selektori.

Koje su poznate verzije CSS jezika kroz razvojni put i koja je aktualna službena verzija?
CSS 1, 2 i 3. CSS3 je aktualna službena verzija.

Koje su glavne karakteristike aktualne službene verzije CSS jezika?
Napredne animacije i tranzicije, višestruke pozadine i gradijenti, višestruki izgledi stupaca, prozirnost, zaobljeni rubovi, selektori,
fleksibilnost fontova, prelamanje riječi, transformirajući elementi i sjenčanje teksta.
https://squareboat.com/blog/css3-features

Navedite jedno jednostavno CSS pravilo sa 2 svojstva.
p {
  text-align: center;
  color: red;
}

Pozicijski na koja tri načina možemo smjestiti i koristiti CSS sa HTML elementima?
Inline, internal i external.
